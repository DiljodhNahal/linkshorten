Diljodh's Link Shortener

Hosted On Heroku, So The Domain Probably Makes It Longer, But You Get The Idea :)

Endpoints

  • GET /:id - Access Existing Shortened Link.
    • Example: https://dil-linkshorten.herokuapp.com/V3BLJuTK
    • Response: Redirect to "https://google.com"
  • POST /shorten/:url - Shorten A New Url.
    • Example: https://dil-linkshorten.herokuapp.com/shorten/htps://google.com
    • Response: {original: "https://google.com", new: "https://dil-linkshorten.herokuapp.com/V3BLJuTK"}
